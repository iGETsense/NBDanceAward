================================================================================
API ENDPOINTS
================================================================================

GET /api/candidates
  Returns all candidates from database
  Response: { success: true, candidates: [...] }

POST /api/candidates
  Add new candidate to database
  Request: { id, name, title, image, category, votes, badge }
  Response: { success: true, candidateId, message }
